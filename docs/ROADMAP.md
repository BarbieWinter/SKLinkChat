# Roadmap

这份路线图描述 SKLinkChat 的公开版本方向。优先级会随着真实使用反馈调整。

## 已完成

- React + Vite 前端应用
- FastAPI 后端服务
- PostgreSQL 数据模型和迁移
- Redis 会话、匹配和在线状态支持
- Stack Auth 登录链路
- 匿名实时聊天
- 举报提交
- 管理后台举报审核
- 管理后台审计日志
- Docker Compose 本地启动
- 非商业使用许可证
- 开源贡献、安全和变更文档

## 正在做

- 更完整的 README 展示内容
- CI 自动检查
- 开源截图和架构说明
- 部署文档整理
- 更稳定的演示数据方案

## 未来计划

- 游客体验模式：不登录也能浏览 UI，但不能真正聊天
- Demo 数据模式：模拟聊天、举报和管理后台数据
- 账号治理增强：封禁原因、限制到期时间、管理员备注
- 用户申诉入口
- 聊天体验增强：正在输入提示、对方离开提示、重新匹配按钮
- 消息发送失败重试
- 基础敏感词过滤
- 更完整的生产部署指南
- 更多端到端验证脚本
